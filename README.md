# PythonProject
I am working to utilize AI and reinforcement learning to allow AI to move a Robotic arm using a camera.
This project is still a work in progress as I have not fully developed the arm just yet so working to create a robotic arm simulator to allow the AI to still make mock movements.
The AI being used is a llama model i got from Hugging Face that is downloaded locally and the robotic arm is a design I found online for 3D printing. More will  be added to read me as project progresses.
